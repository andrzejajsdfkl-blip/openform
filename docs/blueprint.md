# **App Name**: OpenForm Studio

## Core Features:

- Architectural Template Builder: Tool for defining custom document types using a rigid set of 50+ technical fields with user-defined labels and logical order.
- Dynamic Form Engine: Automatically renders production-ready forms from JSON schemas, featuring real-time validation for dates, decimals, and requirements.
- AI Template Architect: A generative tool used to analyze a document name and propose the most logical mapping of technical fields to semantic labels.
- Visual Sort Interface: Drag-and-drop workflow to reorder technical field sequence, which automatically updates the underlying 'order' property in the schema.
- Universal Format Pipeline: Import and export system for the '.off' JSON standard, ensuring full portability of definitions and values across different installations.
- Centralized Record Vault: Database system to store and query documents, with dynamic filtering based on the generated document types and priority labels.
- Base64 Asset Suite: Attachment handling module that supports PDF, images, and Office file previews by processing data directly from Base64 payloads.

## Style Guidelines:

- Primary color: Sophisticated Indigo Blue (#7979D3), providing a stable and professional aesthetic for technical workflows.
- Background color: Deep Slate Charcoal (#16161D), emphasizing high contrast and a modern, high-end agency vibe.
- Accent: Crisp Azure Blue (#5295E1) for calls to action, ensuring prominence without breaking the unified indigo spectrum.
- Headlines: 'Space Grotesk' for its precision-engineered and scientific character; Body: 'Inter' for a clean, neutral, and highly readable look.
- An architectural grid system featuring generous spacing and a technical sidebar for filtering document categories.
- Micro-interactions include smooth layout transitions when reordering fields and subtle pulses during JSON processing.
- Sharp, geometric line icons to represent different document states and data types.
